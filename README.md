# 📒 Smart Notes · 精炼笔记生成器

一个帮你把学习材料快速整理成结构化笔记的 Claude Skill。

---

## 🎯 这个 Skill 是做什么的

学习的时候最头疼的事：

- Claude 帮你总结，结果输出了一篇比原文还长的文章
- 重要知识点埋在大段废话里，根本找不到
- 笔记没有结构，复习的时候不知道从哪看起

**Smart Notes** 就是为了解决这个问题。

你把任何学习材料丢进来（课件、文章、教材片段、录音文字），它会严格按照固定模板输出结构清晰的笔记——重要内容一个不漏，废话一句不留。

---

## 📌 输出结构

每次整理都按这个模板输出：

```
## 🎯 一句话核心
（这份材料在说什么，15字以内）

## 📌 关键概念
- 概念A：一句话解释
- 概念B：一句话解释
（只列最重要的，每条独立完整）

## 🔗 逻辑关系
（说清楚概念之间怎么连的，3句话以内）

## ❓ 容易混淆的点
- 误区：XXX → 正确理解：XXX

## 💡 记忆口诀
（可选，帮助快速记忆核心内容）
```

---

## 🚀 使用方式

直接把学习材料发给 Claude，说：

> 「帮我把这段整理成笔记」
> 「总结一下这篇文章」
> 「做一下笔记」

Skill 会自动触发，按模板输出。

## 📂 文件结构

```

```

---

## 💡 设计思路

这个 Skill 的核心原则只有一条：

> **重要内容必须完整保留，删掉的只是废话。**

不是压缩笔记，不是只抓核心——而是把原始材料里所有有价值的信息，用最简洁的方式重新表达出来。

---

## 🛠 适用材料类型

| 材料类型 | 适用场景 |
|---------|---------|
| 教材 / 课件 | 定义、原理、公式整理 |
| 新闻 / 文章 | 事件、观点、结论提炼 |
| 讲座 / 录音文字 | 核心论点、规律总结 |
| 论文摘要 | 研究问题、方法、结论 |
